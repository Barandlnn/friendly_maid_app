# Friendly Maid App - Flutter UI Implementation

This project is a Flutter implementation of the **Friendly Maid App** design provided in Figma.

The main goal of this project is to convert the given mobile app design into a clean and functional Flutter UI.

## Project Overview

Friendly Maid App is a smart home and food management mobile application UI.  
The app focuses on inventory tracking, expiry alerts, recipe suggestions, shopping assistance, and product management screens.

## Features Implemented

- Splash and welcome screens
- Onboarding screens
- Authentication UI
  - Sign up
  - Sign in
  - Forgot password
  - Reset password
- Home dashboard
- Device connection screen
- Inventory screen
- Product details screen
- Expiry alert detail screen
- Notifications screen
- Recipe suggestions screen
- Recipe details screen
- Buy Now / shopping screen
- Other Products screens
- Menu screen
- Log out confirmation screen
- Delete account confirmation screen
- Web cursor interaction support for clickable UI elements

## Tech Stack

- Flutter
- Dart
- Material Design
- Google Fonts
- Figma-based UI implementation
